# 00. 全体概要

## アプリ名
- **QuotaWatch**

## 目的
SwiftBarプラグインで実装している「クォータ監視」を、macOSネイティブのメニューバー常駐アプリに置換する。

MVPは **Z.ai（GLM）** のクォータ取得を対象とするが、将来的に他プロバイダ（OpenAI / Anthropic / Google / etc.）へ拡張できるよう、最小限の抽象（Providerプロトコル）を採用する。

## 置換の主目的
1. 通知を“アプリとして”扱い、通知設定（許可/サウンド/表示形式）の統制を容易にする
2. `jq/curl/python3/osascript` 等の外部依存を排除し、運用を安定化
3. UI（設定・グラフィカル表示）を追加しやすくする

## 想定ユーザー
- 単一ユーザー（自分用）
- 常時ログインしているmacOS環境

## 非目標（当面）
- App Store配布
- 署名・公証・自動更新（Sparkle等）
- 複数端末同期
- 多プロバイダ対応のUI（MVPでは内部抽象のみ）
